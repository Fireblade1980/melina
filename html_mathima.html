HTML — Βασική Δομή
📄Η δομή κάθε HTML σελίδας

Κάθε ιστοσελίδα έχει μια συγκεκριμένη δομή. Σαν να χτίζεις σπίτι — υπάρχει σκελετός που πάντα υπάρχει!

Ιδού τα 4 βασικά μέρη που βλέπεις σε κάθε HTML αρχείο. Παρατήρησε ότι τα περισσότερα tags έχουν άνοιγμα <tag> και κλείσιμο </tag> — θα μάθεις σύντομα ποια ΔΕΝ κλείνουν!

<!-- 1. Δηλώνουμε ότι είναι HTML5 --> <!DOCTYPE html> <!-- 2. Το "περίβλημα" όλης της σελίδας --> <html lang="el"> <!-- 3. Η "αόρατη" κεφαλίδα --> <head> <title>Η σελίδα μου</title> </head> <!-- 4. Εδώ μπαίνει ό,τι βλέπει ο χρήστης --> <body> <h1>Γεια σου κόσμε!</h1> <p>Αυτό είναι κείμενο.</p> </body> </html>
1️⃣ <!DOCTYPE html>
Λέει στον browser "αυτό είναι HTML5". Πάντα πρώτο!
2️⃣ <html>
Το "κουτί" που περιέχει ολόκληρη τη σελίδα.
3️⃣ <head>
Πληροφορίες για τη σελίδα — δεν φαίνονται στον χρήστη!
4️⃣ <body>
Ό,τι υπάρχει εδώ φαίνεται στην οθόνη!
HTML — Τύποι Tags
🔀Tags που κλείνουν vs tags που ΔΕΝ κλείνουν

Στην HTML υπάρχουν δύο είδη tags. Πολύ σημαντικό να τα ξέρεις!

✅ Tags που ΚΛΕΙΝΟΥΝ
Έχουν άνοιγμα και κλείσιμο. Το κλείσιμο έχει / μπροστά.
<h1>Τίτλος</h1> <p>Κείμενο</p> <b>Έντονο</b> <a>Link</a>
⚡ Tags που ΔΕΝ κλείνουν
Μπαίνουν μόνα τους — δεν έχουν περιεχόμενο μέσα.
<img src="..."> <br> <hr> <input type="text"> <meta charset="UTF-8">

Σκέψου το έτσι: αν το tag περιέχει κάτι (κείμενο, άλλα tags...) χρειάζεται κλείσιμο. Αν είναι μόνο του (εικόνα, γραμμή) — δεν κλείνει!

<p>Ο σκύλος μου λέγεται Rex.</p> <-- κλείνει ✅ <br> <-- ΔΕΝ κλείνει ⚡ <img src="rex.jpg" alt="Rex"> <-- ΔΕΝ κλείνει ⚡ <p>Είναι πολύ χαρούμενος!</p> <-- κλείνει ✅
💡 Το <br> κάνει αλλαγή γραμμής και το <hr> βάζει οριζόντια γραμμή — και τα δύο δεν κλείνουν!
HTML — <head> & <title>
🏷️Τι μπαίνει στο <head>;

Το <head> είναι σαν το "παρασκήνιο" — δεν φαίνεται στην οθόνη, αλλά ο browser το χρησιμοποιεί!

<head> το όνομα στην καρτέλα του browser: <title>Η σελίδα του Γιώργου</title> η γλώσσα των χαρακτήρων (ελληνικά κ.τ.λ.): <meta charset="UTF-8"> σωστό μέγεθος στο κινητό: <meta name="viewport" content="width=device-width"> </head>
Το <title> φαίνεται εδώ ↓
🌐 Η σελίδα του Γιώργου
Εδώ φαίνεται το περιεχόμενο του <body>...
💡 Το <title> φαίνεται στην καρτέλα του browser και στα αποτελέσματα του Google!
HTML — Τίτλοι
🔤Τίτλοι: h1 ως h6

Υπάρχουν 6 επίπεδα τίτλων! Το h1 είναι ο μεγαλύτερος — σαν τον τίτλο βιβλίου. Το h6 είναι ο πιο μικρός — σαν υποσημείωση.

<h1> Κύριος τίτλος
<h2> Ενότητα
<h3> Υποενότητα
<h4> Μικρότερος τίτλος
<h5> Ακόμα μικρότερος
<h6> Ο πιο μικρός
<h1>Η σελίδα μου</h1> <h2>Τα χόμπι μου</h2> <h3>Ποδόσφαιρο</h3> <h4>Η αγαπημένη μου ομάδα</h4>
💡 Χρησιμοποίησε μόνο ένα <h1> ανά σελίδα — είναι σαν το όνομα της σελίδας! Τα h2–h6 μπορείς να τα βάλεις όσα θέλεις.
HTML — Κείμενο
✍️Παράγραφοι & κείμενο

Το tag <p> φτιάχνει παραγράφους. Κάθε <p> αρχίζει σε νέα γραμμή με κενό πάνω-κάτω.

Μέσα στο <p> μπορούμε να κάνουμε κείμενο έντονο ή πλάγιο:

<p>Αυτή είναι η πρώτη παράγραφος.</p> <p>Εδώ έχω <b>έντονο</b> και <i>πλάγιο</i> κείμενο.</p> <p>Τρίτη παράγραφος — αυτόματο κενό πάνω!</p>
Αποτέλεσμα στον browser

Αυτή είναι η πρώτη παράγραφος.

Εδώ έχω έντονο και πλάγιο κείμενο.

Τρίτη παράγραφος — αυτόματο κενό πάνω!

⚠️ Αν απλά πατήσεις Enter στον κώδικα, ο browser δεν κάνει νέα γραμμή! Πρέπει να χρησιμοποιήσεις <p> ή <br>.

CSS
🎨Τι είναι η CSS;

Η CSS (Cascading Style Sheets) δίνει ομορφιά! Αν η HTML είναι ο σκελετός, η CSS είναι τα χρώματα και η διακόσμηση.

h1 { color: blue; font-size: 32px; text-align: center; } p { background-color: yellow; font-family: Arial; padding: 10px; }
Αποτέλεσμα

Τίτλος σε μπλε!

Κείμενο με κίτρινο φόντο!

💡 Το CSS μπαίνει μέσα σε <style> στο <head>, ή σε ξεχωριστό αρχείο .css
JavaScript
⚡Τι είναι η JavaScript;

Η JavaScript (JS) δίνει ζωή! Κάνει πράγματα να κινούνται, να αλλάζουν και να ανταποκρίνονται στον χρήστη.

HTML = σώμα, CSS = ρούχα, JS = κινήσεις και λόγια!

// Εμφανίζει μήνυμα alert("Γεια σου, κόσμε!"); // Αλλάζει χρώμα κειμένου function αλλαξέΧρώμα() { document.getElementById("τίτλος") .style.color = "red"; }
Δοκίμασέ το!

🌈 Γεια σου, κόσμε!
